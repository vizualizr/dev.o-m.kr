- ## Troubleshooting
	- **의존성 붕괴 및 package.json 복구**
		- **현상**: `astro add` 도중 패키지 매니저 충돌로 `package.json` 파손 및 스크립트 소실.
		- **조치**: Git 이력을 추적하여 원본 복구 후 `npm install`을 통한 수동 패키지 통합 및 정합성 확보.
	- **Content Schema 매핑 오류**
		- **현상**: `InvalidContentEntryDataError` 발생으로 인한 빌드 중단.
		- **조치**: `content.config.ts` 내 로더에서 구형 데이터 타입을 신규 타입(`document`, `letter`, `poster`)으로 변환하는 타입 매핑 로직 추가로 해결.
	- **Tailwind CSS v4 @theme 인식 오류**
		- **현상**: VS Code 및 빌드 과정에서 `@theme` 규칙이 인식되지 않거나 경로 해결 실패.
		- **조치**: CSS import 순서 최적화(상단 배치), alias 제거 후 상대 경로 사용, VS Code Custom CSS Data 설정을 통해 해결. [[DONE]]
