- ## Summary
- ## Steps
	- 외부 리소스를 비동기적으로 불러오는 DOM 요소에 존재하는 프로퍼티이다.
	- 해당 DOM 요소가 외부 자원을 모두 불러오면 지정된 함수를 실행한다.
		- `window.onload`: 웹 페이지의 모든 자산을 온전히 불러온 뒤 지정된 함수를 실행한다.
		- `<body>, <embed>, <iframe>, <img>, <link>, <object>, <script>, <style>, <track>`
